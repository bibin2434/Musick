# Musick

Musick is a fun and interactive web-based music instrument simulator that allows users to play various musical sounds by clicking buttons or pressing corresponding keys on the keyboard.

## Features

- Play different musical instrument sounds using the keyboard or on-screen buttons.
- Includes sounds for accordion, drum, electric guitar, flute, piano, harmonica, saxophone, and violin.
- Visual button animations to enhance user interaction.

## How to Use

1. **Button Click:** Click on any of the buttons labeled 1 through 8 to play the corresponding instrument sound.
2. **Keyboard Press:** Press the keys 1 through 8 on your keyboard to play the same sounds as the buttons.
3. The button associated with the sound will animate when clicked or when the corresponding key is pressed.

## Sound Files

The project uses the following sound files:
- `accordion.mp3`
- `drum.mp3`
- `elec guitar.wav`
- `flute.wav`
- `piano.mp3`
- `harmonica.mp3`
- `sax.mp3`
- `violin.wav`

Make sure these files are placed in a `sound/` directory within the project.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/bibin2434/Musick.git
   ```

2. Open `index.html` in your browser to start playing.
